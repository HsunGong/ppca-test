problem

Toad Mikhail has an array of 2k2^k2k integers a1,a2,⋯ ,a2ka_1,a_2, \cdots ,a_{2^k}a1​,a2​,⋯,a2k​.

Find two permutations ppp and qqq of integers 0,1,⋯ ,2k−10, 1, \cdots ,2k−10,1,⋯,2k−1, such that aia_iai​ is equal to pi⊕qip_i \oplus q_ipi​⊕qi​ for all possible iii, or determine there are no such permutations. Here ⊕\oplus⊕ denotes the bitwise XOR operation.

Input

The first line contains one integer kkk (2≤k≤122 \leq k \leq 122≤k≤12), denoting that the size of the array is 2k2^k2k.

The next line contains 2k2^k2k space-separated integers a1,a2,⋯ ,a2ka_1,a_2, \cdots ,a_{2^k}a1​,a2​,⋯,a2k​ (0≤ai<2k0 \leq a_i < 2^k0≤ai​<2k) — the elements of the given array.

Output

If the given array can't be represented as element-wise XOR of two permutations of integers 0,1,⋯ ,2k−10,1,\cdots,2^k−10,1,⋯,2k−1, print "Fou".

Otherwise, print "Shi" in the first line.

The next two lines should contain the description of two suitable permutations. The first of these lines should contain 2k2^k2k space-separated distinct integers p1,p2,⋯ ,p2kp_1,p_2,\cdots,p_{2^k}p1​,p2​,⋯,p2k​, and the second line should contain 2k2^k2k space-separated distinct integers q1,q2,⋯ ,q2kq_1,q_2,\cdots,q_{2^k}q1​,q2​,⋯,q2k​.

All elements of ppp and qqq should be between 000 and 2k−12^k−12k−1, inclusive; pi⊕qip_i \oplus q_ipi​⊕qi​ should be equal to aia_iai​ for all iii such that 1≤i≤2k1 \leq i \leq 2^k1≤i≤2k. If there are several possible solutions, you can print any.
